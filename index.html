Hosting on GitHub Pages: Step-by-Step

  1. Create a new repository on GitHub named "username.github.io" (replace "username" with your GitHub username)
  2. Create an index.html file in your repository (this is the file you're viewing right now!)
  3. Paste your HTML code (this entire page) into the index.html file
  4. Commit your changes to the repository
  5. Go to Settings > Pages and set the source branch to "main" (or "master")
  6. Wait a few minutes for GitHub to build your site
  7. Visit your new site at https://username.github.io
